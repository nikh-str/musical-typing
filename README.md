# musical-typing
typing test written in rust

based on [typr](https://github.com/Sakura-sx/typr) and using [gum](https://github.com/charmbracelet/gum).


![demo](assets/intro.png)
![demo2](assets/main.png)



```
git clone https://github.com/nikh-str/musical-typing.git
cd musical-typing
cargo run --release
```
